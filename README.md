<h1>Microservices Project API Gateway</h1>

This repository contains a Spring Boot Microservices Architecture implementation using Spring Cloud, Eureka, Feign Client, and API Gateway.
The goal of this project is to understand service-to-service communication, service discovery, and centralized routing in a real-world microservices setup.

<b>Project Structure</b>
```
microservices
├── api-gateway
├── discovery-server
├── user-service
└── task-service

```
<b>Technologies Used</b>
Java 21
Spring Boot 3.x
Spring Cloud 2025.x
Spring Cloud Gateway
Eureka Server & Client
OpenFeign
Spring Data JPA
H2 / MySQL (configurable)
Maven
